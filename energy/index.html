          Mobile Rig Digital Experience Transformation

          What: Aligning with client's digital transformation initiative with respect to developing the live Mobile Rig application, 
          an iOS application was created for Information Visibility and Operational Efficiency improvement using real time Rig Data. 
          It also incorporated real time alerts to inform Rig Supervisor of ongoing anomalies and corrective actions to be taken for the alerts.

          Why: The unstructured data collected from sensors on Rigs did not have an easy interpretation and by the time the IT team got the details from
          collected data, it would have been too late to identify an event There was a need for a digital app which could understand the unstructured data,
          derive Insights out of it and alert the Rig Supervisors.It also had a ML algorithm to predict any failures based on past failure/type data.

          Technologies Used: Swift, Azure SQL, Azure APIM and App Services, Python, Azure ML
